Pipes / Custom pipes

Custom Pipes
Default image

This custom pipe named 'defaultimage', which is used if the actual image is not present



Capitalize first character

Custom pipe named 'firstcharcateruppercase', which is used to capitalize first character in the string. Here 'rickky' and 'bret' are passed.

  • {{obj.firstName | firstcharcateruppercase}} {{obj.lastName}}
Truncate characters

Custom pipe that display the first 'n' (50, 100, 150, ...) characters of the description

Result: {{truncateString | truncate: 150}}



Simple Multiplication

In this custom pipe, it will go through all the items in the array and multiply each item with specified number. multiplyArray = [2, 3, 4, 5, 6]; and multiply with 100

  • {{val}} * {{multiplyBy}} = {{val | multiply: multiplyBy}}

  • {{company.companyName}} - {{company.primaryExchange}}

Built-in Pipes

Date

Today - {{today | date:"MM/dd/yy"}}

Timestamp: 1382086394000 -> {{timestamp}}

   =====> The hero's birthday is {{ birthday | date:format }}

The chained hero's birthday is: {{birthday | date:format | uppercase}}

Decimal

Decimal tobe displayed for 3.14159265 using number: '3.1-2':    {{decimalValue | number: '3.1-2'}}

Decimal tobe displayed for 3.14159265 using number: '1.4-4':    {{decimalValue | number: '1.4-4'}}

Lowercase / Uppercase

lowercase => {{"This transforms a string to lowercase" | lowercase}}

uppercase => {{"This transforms a string to uppercase" | uppercase}}

Slice

This returns a slice of an array. The first argument is the start index of the slice and the second argument is the end index.

Sliced for the array [1, 2, 3, 4, 5, 6, 7] using slice:1: 4 :    {{[1, 2, 3, 4, 5, 6, 7] | slice:1: 4}}

Sliced for the array [1, 2, 3, 4, 5, 6, 7] using slice:2 :    {{[1, 2, 3, 4, 5, 6, 7] | slice: 2}}

Sliced for the array [1, 2, 3, 4, 5, 6, 7] using slice:2:-1 :    {{[1, 2, 3, 4, 5, 6, 7] | slice: 2:-1}}

Currency

Format numbers in CAD Currency:    {{price | currency:'CAD':'symbol-narrow'}}

Format numbers in USD Currency:    {{price | currency:'USD':'code'}}

Format numbers in EUR Currency:    {{price | currency:'EUR':'symbol'}}

Json

Useful for debugging and displays an object as a Json string: "[{{dummyObj | json}}]"

{{dummyObj | json}}

Percent

Format numbers (0.123456) in percentage using 'percent' pipe:    {{0.123456 | percent}}

Format numbers (0.123456) in percentage using "percent: '2.1-2'":    {{0.123456 | percent: '2.1-3'}}

Format numbers (42) in percentage using "percent: '10.4-4'":    {{42 | percent: '10.4-4'}}

AsyncPipe

This pipe accepts an observable or a promise and lets us render the output of an observable or promise without having to call then or subscribe.
